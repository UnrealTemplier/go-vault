# 🧭 Общая идея системы

Все задачи разбиты не по сложности, а по **типу мышления**, которое они формируют.

1. Понять паттерн  
2. Решить базовые задачи  
3. Научиться распознавать вариации  
4. Перейти дальше  
5. Вернуться позже и добить сложные задачи

# 🧠 КАК ПРАВИЛЬНО ПРОХОДИТЬ ЭТУ СИСТЕМУ

- уметь объяснять решение вслух

---
# 📊 ВНУТРЕННЯЯ ЛОГИКА СИСТЕМЫ

Каждая следующая фаза опирается на предыдущую:

- массивы → окна → состояние  
- стек → дерево решений  
- графы → пространство состояний  
- DP → обобщение всех предыдущих
